## 内容定位

账号不是“房源展示型”，而是“研究判断型”。核心承诺不是提供更多房源，而是帮助用户降低决策成本：

- 把重庆板块和项目排出优先级
- 对价格、政策和市场说法做判断
- 把土拍、轨道和宏观信息翻译成购房影响
- 用系列内容积累方法论资产
- 用情绪和人生议题扩大房产内容的受众入口

“断档级领先”等简介表达强化权威感，但普通 KOS 不应照搬无法证明的能力宣称。

## 选题结构

### 主体内容

- 板块优先级：礼嘉、中央公园东区、西区、大坪
- 项目深度研判：定价、开盘、户型和园林
- 政策与宏观：文件解读、补贴、《求是》、楼市趋势
- 城市发展：土拍、轨道、核心地块
- 决策教育：抗跌逻辑、贷款方式、下跌焦虑

### 流量补充

- 人群情绪：女生婚前买房
- 参与型话题：众筹拿地、期待哪块地
- 资料型内容：户型图、动态地图、全资料

## 可复制的方法

### 1. 用“判断”代替“介绍”

标题中频繁出现“优先级、研判、错误解读、秘诀、最合理”等判断词。用户点击的原因不是了解项目存在，而是想知道“应该怎么选”。

复制原则：每篇先形成一句核心判断，再决定需要哪些资料支撑。

### 2. 把复杂主题做成系列

礼嘉拆成篇一、篇二、下篇；抗跌逻辑做成连续系列；板块优先级形成固定栏目。

复制原则：一个大主题拆为“框架篇、区域篇、项目篇、决策篇”，降低单篇信息密度并形成追更。

### 3. 同时经营深度与及时性

深度内容包括板块优先级、定价研判；及时内容包括新政、土拍、即将开盘、户型资料。

复制原则：内容排期采用“常青深度 + 当周热点 + 实用资料”组合，避免只追热点或只写长文。

### 4. 重庆地域信号强

标题高频出现重庆、礼嘉、中央公园、照母山、两江新区、大坪等词，搜索意图明确。

复制原则：标题和正文都要出现具体城市、板块、预算或人群，不写泛泛的全国楼市评论。

### 5. 资料后面跟判断

户型图、土拍和轨道本身是信息；真正形成账号差异的是对价格、适配人群和决策影响的解释。

复制原则：资料型内容至少增加“怎么看、影响谁、下一步核实什么”三部分。

### 6. 专业内容混入人群议题

“女生婚前买房”“害怕继续跌怎么办”让账号不只面对正在看某个盘的人，也触达有居住决策焦虑的人。

复制原则：专业内容与场景内容约按 7:3 配置，场景内容仍需给出方法，不做空泛情绪。

## 风险与改进

- 强判断标题容易滑向绝对化，必须补充适用边界。
- 政策、价格、项目状态变化快，必须注明信息日期。
- “抗跌、最合理”等表达需要展示评价维度，不能变成收益承诺。
- 教育、轨道和规划内容要区分已兑现、在建、规划和传闻。
- 资料或户型图应确认来源和使用权限。
- 不复制账号名称、个人身份、简介承诺、固定栏目原句或具体文章。

## 方法论结论

该账号最值得复用的不是某个标题，而是这条内容链：

`本地信息 -> 明确判断 -> 系列栏目 -> 证据解释 -> 风险边界 -> 用户行动`

Skill 应把这条链标准化，同时用事实核验和合规检查约束强判断。
