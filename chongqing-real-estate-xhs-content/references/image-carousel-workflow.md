# 小红书正文图片生成流程

## 适用场景

当用户要求生成“小红书图片、正文图、轮播图、封面图、图文长图、直接做成图片”时使用本流程。先用主 Skill 完成内容判断、分页和合规检查，再生成图片。

默认输出为 3:4 竖版 PNG，适合小红书手机阅读。

## 默认协作流程

1. 先把笔记内容拆成 6 至 8 页轮播脚本。
2. 生成第 1 页封面样图。
3. 等用户确认视觉方向。
4. 确认后，用同一视觉系统生成剩余页面。
5. 将全部图片复制到工作区独立文件夹。
6. 检查文件数量、命名、尺寸、页码和明显文字错误。

如果用户明确要求“直接全套生成”或“不要等确认”，可以跳过第 2 至 3 步，一次生成整组图片，但仍需最终检查。

## 统一视觉风格

默认视觉系统：

- 比例：3:4 竖版
- 背景：暖白色轻纸张质感
- 主色：深海军蓝
- 重点色：重庆暖橙
- 辅助色：灰蓝
- 风格：研究报告感 + 小红书亲和感
- 元素：极淡重庆山城等高线、楼宇、轨道、桥梁、坡坎或跨江线稿
- 字体：端正中文黑体风格，层级清晰，主标题最大
- 版式：清晰网格、圆角卡片、清单、时间轴、流程图和提示框

避免：

- 大红大绿、高饱和渐变、豪宅广告感
- 房价暴涨暴跌箭头、金币、K 线、恐慌视觉
- 二维码、品牌 Logo、无授权项目图
- 密集小字、艺术变形字、过度装饰

## 每页信息密度

每页只表达一个核心意思。

推荐结构：

- 小标签：4 至 8 个字
- 主标题：8 至 18 个字
- 正文：2 至 5 条短句
- 结论或提示：1 句
- 页码：`第X页 / 共Y页`

小红书图片最怕文字太多。若一页超过 120 个汉字，优先拆页，而不是缩小字号。

## 8页决策型轮播模板

1. 封面：核心问题和主结论
2. 核心结论：适合谁、不适合谁
3. 判断维度一：例如持有周期
4. 判断维度二：例如现金流
5. 判断维度三：例如重庆真实通勤和兑现状态
6. 可以行动的人群
7. 需要先等等的人群和风险边界
8. 行动清单、互动问题、信息说明

## 图片生成提示词骨架

每页单独生成，保持同一视觉系统。提示词必须包含：

```text
Use case: infographic-diagram
Asset type: 小红书3:4竖版图文轮播第X页
Primary request: 延续已确认的重庆房产知识型账号视觉风格，制作第X页，主题是【本页主题】。
Scene/backdrop: 暖白色轻纸张质感，极淡重庆山城等高线与城市线稿底纹，可包含楼宇、轨道、桥梁、坡坎等抽象元素。
Style/medium: 高品质中文商业信息图，研究报告感与小红书亲和感结合，扁平化编辑设计，清晰网格和充足留白。
Composition/framing: 3:4竖版。说明本页具体布局。
Color palette: 暖白底，深海军蓝主文字，重庆暖橙重点，少量灰蓝辅助。
Text (verbatim):
逐行列出所有文字。
Typography: 中文黑体风格，粗细层级明确，字形端正，所有文字必须逐字准确、清晰可读。
Constraints: 严格3:4；信息层级清楚；边缘保留安全距离；不出现人物、二维码、账号名、价格、收益承诺、品牌Logo或水印。
Avoid: 文字乱码、错别字、密集小字、房价暴涨暴跌箭头、金币、夸张营销、恐慌氛围、复制任何现有博主版式。
```

## 封面样图标准

第 1 页封面要先确认：

- 主标题是否醒目
- 中文是否清晰无错
- 是否符合重庆房产研究型账号气质
- 是否不制造焦虑
- 是否能作为后续页面的统一风格母版

用户确认后，后续页面必须沿用：

- 同一底色
- 同一主色和强调色
- 同一页码格式
- 同一图标线稿语言
- 同一标题和卡片层级

## 文件归档

默认在当前工作区创建独立文件夹，文件夹名使用主题，例如：

```text
重庆买房怕继续跌-小红书轮播图
```

文件命名：

```text
01-封面.png
02-核心结论.png
03-持有周期.png
04-家庭现金流.png
05-重庆真实通勤.png
06-适合买房条件.png
07-适合继续等待.png
08-行动清单.png
```

按实际主题改名，但必须保留两位数字顺序。

## 验收检查

交付前检查：

- 图片数量是否等于脚本页数
- 每张是否为 3:4 竖版
- 每张是否为 PNG
- 文件是否已复制到工作区或用户指定目录
- 页码是否连续且总页数一致
- 主标题是否清楚
- 是否有明显错别字、乱码、重复字或截断
- 是否出现价格、收益承诺、教育承诺、违规金融或虚假稀缺
- 是否与封面样图风格统一

如果发现某页明显错误，只替换该页，不重做整套。

## 完成回复

按项目规则简短回复：

```text
已完成。

本次改动：
1. 生成完整 X 页小红书轮播图。
2. 统一为 3:4 竖版。
3. 已按发布顺序完成命名。

验证结果：
1. X 张图片齐全。
2. 尺寸、页码和视觉风格一致。
3. 未加入高风险房产营销表达。

文件位置：[文件夹名](/绝对路径)
```
