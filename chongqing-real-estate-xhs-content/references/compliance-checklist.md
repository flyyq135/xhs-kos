# 重庆房产小红书合规检查

## 发布前必查

### 事实

- 价格、首付、利率、税费和补贴是否注明日期与口径
- 项目在售、待售、售罄和交付状态是否核实
- 轨道、道路、学校、商业和医院是否区分已建、在建、规划和传闻
- 通勤时间是否说明出发点、时段和交通方式
- 数据是否有来源，样本量是否足够
- 户型图、地图、项目效果图是否注明来源或获得使用许可

### 数据表达

允许做分析型数据表达，但必须在内部底稿写清口径。正式笔记中默认不堆来源链接和平台出处，要把数据转化为经纪人的专业判断：

- 挂牌价：注明平台、采样日期、面积段或户型范围。
- 成交价：注明成交时间范围、样本数量和是否同户型/同楼栋可比。
- 成交周期：注明统计口径，不把个别房源当整体趋势。
- 租金、车位、物业费：注明来源和时间，无法确认时标待核实。
- 板块/小区对比：必须说明比较对象和比较维度。

禁止：

- 用单套成交代表整个小区或板块走势。
- 用单个挂牌价制造“便宜/贵”的结论。
- 把未核实的价格、成交、学位、规划写成确定事实。
- 把“近90天成交少”直接写成“没人买”，除非有充分数据和口径。
- 把“挂牌价下降”写成“房价大跌”，应改为“部分挂牌价格有调整，需结合成交核实”。
- 在正式笔记末尾堆来源链接，导致内容像资料搬运而不是经纪人研判。
- 因为不展示来源，就把数据全部删掉，导致内容没有数字支撑。

教育、价格、成交、学位和规划必须在内部强制标记来源或待核实。正式笔记中涉及高风险事实时，用“以当年官方政策为准”“下定前核实同户型成交口径”等客户能理解的说法表达。

正式笔记可以出现经过转化的数据锚点，例如“十多年楼龄”“近阶段同户型成交”“步行约几分钟”“参考价差一截”。数据必须服务判断，不能制造收益、稀缺或恐慌。

### 房产表达

禁止或改写以下高风险表达：

| 不建议 | 稳妥表达 |
|---|---|
| 捡漏 | 适合重点比较 |
| 大降价 | 当前价格较前期有调整或沟通空间 |
| 零首付 | 删除，不做此类宣传 |
| 首付贷、经营贷买房 | 删除，不提供违规融资建议 |
| 稳赚不赔 | 删除 |
| 包成交 | 提升匹配和沟通效率 |
| 内部房源、内部消息 | 以公开信息或可核实渠道为准 |
| 必涨、保值、抗跌神盘 | 讨论影响价格的条件和风险 |
| 学区承诺 | 周边教育资源以当年主管部门政策为准 |
| 最后一套、马上没了 | 仅在库存可核实且注明时间时客观描述 |

### 判断边界

- 不把“可能、规划、预期”改写成“确定、必然、即将兑现”
- 不用单一成交案例代表整个板块
- 不把个人偏好包装成普遍结论
- 不隐去楼层、朝向、噪音、物业、车位、供应量等明显风险
- “优先级、合理、抗跌”等词后必须给评价维度和适用条件

### 教育与金融

- 不承诺入学、学位、划片稳定或政策永久不变
- 不提供绕监管贷款、垫资、首付贷、经营贷购房方案
- 贷款比较说明利率、期限、提前还款和现金流假设
- 不把财务测算写成个性化投资收益承诺

### 平台与转化

- 不诱导私下加微信或规避平台规则
- 不制造恐慌、羞辱未买房者或唱衰市场
- 不虚构客户案例、排队、抢购或稀缺性
- 不冒充政府、研究机构、开发商或其他创作者
- 不复制他人标题、正文、图表和固定栏目包装

## 合规输出格式

在交付末尾附：

```text
合规自检：
- 动态事实：已核实 / 待核实
- 规划与兑现：已区分
- 收益与涨价承诺：无
- 教育承诺：无
- 违规融资建议：无
- 虚假稀缺或私下引流：无
- 他人内容复刻：无
```

如有待核实项，明确列出并阻止把该项写成确定结论。
