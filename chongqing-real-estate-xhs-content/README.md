# 重庆房产小红书内容 Skill

这是一个面向重庆房产经纪人、KOS、顾问和机构账号的 Codex Skill，用来策划和生成研究型小红书内容。

它适合处理：

- 账号定位和内容方向梳理
- 重庆板块、楼盘、小区研判
- 政策、土拍、轨道、商业和城市规划解读
- 买房决策科普、避坑清单和对比分析
- 小红书图文笔记、3:4 轮播页和正文图片脚本
- 短视频口播脚本、封面文案和月度内容日历
- 内容复盘、选题库和经纪人执行流程

## 核心特点

- 重庆本地化：关注坡坎、跨江、轨道换乘、板块兑现、供应结构和真实看房动线。
- 结论先行：先给普通购房者能理解的判断，再展开证据和行动建议。
- 数据支撑：要求价格、成交、库存、通勤、楼栋、户型等关键信息有口径或待核实提示。
- 实地体验：强调噪音、停车、出入口、坡坎、商业人流等现场观察。
- 风险并列：不只写优点，也写适用人群、不适用人群和风险边界。
- 合规表达：避免承诺涨价、收益、入学、政策结果或制造焦虑。

## 适合谁用

- 重庆房产经纪人
- 区域顾问或门店负责人
- 房产类 KOS / KOC
- 机构新媒体运营
- 想把房产资料转成小红书内容的人

## 不适合什么

- 不适合直接生成夸张营销稿。
- 不适合在缺少事实资料时编造价格、政策、学校、规划或成交数据。
- 不适合复刻某个具体博主的人设、原文、口头禅或个人经历。

## 基本使用方式

把整个 `chongqing-real-estate-xhs-content` 文件夹放到 Codex Skills 目录后，在 Codex 中提出任务即可。

示例：

```text
用 chongqing-real-estate-xhs-content 帮我做一篇重庆礼嘉板块的小红书选题和正文，面向改善家庭，先出逻辑计划。
```

```text
帮我把这个楼盘资料整理成小红书 3:4 轮播脚本，要求结论先行，带风险提示。
```

```text
帮我做一个重庆买房避坑系列的 30 天选题日历。
```

## 推荐工作流

1. 先说明目标客户、区域或主题。
2. 让 Skill 先输出逻辑计划和选题方向。
3. 确认主选题、标题和封面方向。
4. 再生成完整正文、口播稿或轮播分页。
5. 最后做事实核验、合规检查和发布前优化。

## 文件说明

- `SKILL.md`：Skill 主说明和执行规则。
- `agents/openai.yaml`：Agent 配置。
- `references/account-analysis.md`：账号方法和内容风格参考。
- `references/agent-workflow.md`：经纪人使用流程。
- `references/content-frameworks.md`：选题、标题和结构模板。
- `references/data-and-fieldwork.md`：楼盘、小区、板块研判的数据和实地检查方法。
- `references/image-carousel-workflow.md`：小红书图片和轮播图工作流。
- `references/compliance-checklist.md`：房产内容合规检查清单。

## 使用提醒

涉及价格、政策、利率、教育、规划、在售状态、成交数据等动态信息时，需要优先核实最新资料。资料不足时，应降低结论强度，并把关键事项列入待核实清单。
